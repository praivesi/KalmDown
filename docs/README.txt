1. Project Name : KalmDown

2. Meaning of "KalmDown"

  => "KalmDown" is an abbreviation of "Count Down" which is the main purpose of this app.
     It also sounds like "calm down" which means relax. It's intended to remind worshiper's minds that
     it's time to calm down and prepare for worship.
  
  => History of naming : "CountDown" -> "CalmDown" -> "KalmDown"

3. Purpose of "KalmDown" 

  => This app is made for counting down before worship starts at Church.
     Because main target ages are 4 ~ 6 th grades of elementary school,
     UIs of this app concentrate on elementary student's interests.
